# Flappy Bird Game

[![Flappy Bird Cover](https://i.ibb.co/S4ZsY1q/flappy-bird-github-cover.png)](https://ibb.co/tTRk1ydX)

A simple Flappy Bird-style game built with **Phaser 3**, **TypeScript**, and **Vite**.

## Demo

- Live: [https://flappy-bird-game-puce.vercel.app/](https://flappy-bird-game-puce.vercel.app/)

## Features

- Smooth bird movement and jump physics
- Endless pipe obstacles
- Score tracking while playing
- Game over and restart flow
- Lightweight setup for fast local development

## Getting Started

### 1) Clone the project

```bash
git clone https://github.com/tkahmedkamal/flappy-bird-game.git
cd flappy-bird-game
```

### 2) Install dependencies

```bash
npm install
```

### 3) Run in development

```bash
npm run dev
```

### 4) Build for production

```bash
npm run build
```

### 5) Preview production build

```bash
npm run preview
```

## Controls

- `Space` / `Mouse Click`: Flap
- `R` (if enabled in your game scene): Restart after game over
